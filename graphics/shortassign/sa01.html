SA 1: Make your own JavaScript-powered web page

At this point, you might be wondering how you can create your own JavaScript program, on a web page outside of the console environment. Luckily this is fairly easy to do!

All you need is a web browser and a text editor. A text editor is an application that can be used to edit plain text files, kind of like a Microsoft Word stripped of all the extra formatting features. If you don't like working with the default text editors that came pre-installed on your computer, more advanced alternatives include Notepad++ and TextMate for Mac.

The HTML base

When you have your text editor ready, make an HTML file named my_first_program.html and copy into it the following text:

<!DOCTYPE html>
<html>
	<head>
	</head>
	<body>
	</body>
</html>

This is the minimum amount of text you will need to include in every HTML page (although it is would only show up as a blank page in your browser as is). You can think of HTML as the base or the skeleton of a web page. It defines the general structure, including sections and headings and paragraphs. Let's take a closer look at the text you just added to your HTML and what it means:

  • HTML is organized by a hierarchy of tags, which are essentially keywords enclosed in angular brackets. Each meaningful unit of content is preceded by a start tag (e.g. <body>) and followed by an end tag with a slash (e.g. </body>)
  • The purpose of the first line <!DOCTYPE html> is to inform your browser of the file type (in this case, HTML)
  • The text between <html> and </html> is the entirety of the web page.
  • The text between <body> and </body> is the static content that will be visible in the browser every time the web page is opened.
  • The text between <head> and </head> may include some tags that load additional content before the main page content loads.

If you'd like to read more on HTML, W3Schools is an excellent resource to check out.

Incorporating the HTML Canvas

We will now include a canvas element called "myFirstCanvas" on your HTML page. Add the following tag between the <body></body> tags in your HTML file:

<canvas id="myCanvas" width="600" height="400"></canvas>

A canvas is a special type of graphics element built-in to the HTML language. When adding a canvas to your page, you should specify a name (id), and size (width and height), as the tag above shows.

Animating the Canvas with JavaScript

Now download processing.js and processing-wrapper.js and save them to the same directory where your my_first_program.html is located. These JS files will enable the use of the ProcessingJS library. ProcessingJS is a selection of additional code that enhances the appearance of the graphics canvas. If you are interested, you can read more about it here.

Since the graphics script we want to include in my_first_program.html depends on ProcessingJS, we must give the browser access to the ProcessingJS code for our animation to work. To incorporate JavaScript into an HTML page, we use the HTML <script> tag. Between the <body> tags of the HTML, and following the <canvas> element, add the following <script> tags:

<script type="text/javascript" src="processing.js"></script>
<script type="text/javascript" src="processing-wrapper.js"></script>

Next download animation.js to the same directory. The code for an animation is written for you, so all you will have to do is to include it in your page as well. Below your Processing script includes, add:

<script type="text/javascript" src="animation.js"></script>

If everything was done correctly, you should be able to open up my_first_program.html in your browser (Just open your browser and select File > Open from the menu) and see a nice colorful animation appear on your page. Congratulations, you are done! Please submit a screenshot of the animation on your page.
