CS1 Windows Installation

This page shows you how to install the CS 1 software on your PC. You will install all of the following:

  • Python 2.7
  • Eclipse IDE
  • PyDev add-on for Eclipse
  • The CS 1 library, including graphics software

These instructions are oriented toward Windows 7. If you are running a different version of Windows (such as XP or Vista), you will see some differences from what this page shows. If you encounter any trouble installing the software, make sure to come to the "Install-Fest," 7:00 pm to 10:00 pm on Tuesday, September 11 in 003 Sudikoff.

Install Python 2.7

Python 2.7 can be downloaded here.

Once it has finished downloading, double-click on the installer to run it. Once the installer is running, click Next three times. This should start the actual installation. Once the installation is complete, click Finish.

Install Eclipse

Eclipse is the name of the software program you will use to type in your Python programs and run them. Download it:

  • Eclipse IDE.

Since the Eclipse download provides a compressed file, extract it by right-clicking on the download, and selecting Extract All... or Extract Here. If you're asked to select a folder to extract to, choose an easily accessed location, such as your Desktop. Click Next or OK whenever you see these buttons, and if you see a Finish button, click it.

A window with a single folder labeled eclipse should pop up. Drag this folder to somewhere on your computer that you can access easily, perhaps your Desktop.

Open the eclipse folder, and run Eclipse. Since this is your first time running eclipse, the following dialog box will pop up:

You will have your name instead of Betty. Check the box that says Use this as the default and do not ask again and click OK.

Once Eclipse finishes loading, you will see the following screen:

Click the icon that says Workbench in the top right. You will never see this screen again.

Install PyDev

The PyDev add-on for Eclipse can be downloaded within Eclipse. To start, run Eclipse. You should see the Eclipse window:

From the menu bar, select Help and then Install New Software....

The following dialog box should pop up:

Click Add... in the top right.

In this dialog box, under Name, type PyDev. Under Location, type http://pydev.org/updates

Click OK on this dialog box. Once Eclipse loads the software list, make sure that PyDev has a check next to it, and then click Next:

(Ignore the red ellipse around Add...; I added that by mistake.) Read and agree to the Eclipse licence, and then click Finish.

While Pydev is installing, the following dialog box may pop up. Check the box next to Aptana PyDev and click OK:

When Pydev has finished installing, click Restart Now to restart Eclipse.

Start at your main hard drive (probably called C:). From there, select Python27, then python.exe. Then click Open, then OK.

At this point, you have a working version of Eclipse that will enable you to run Python programs. But you cannot yet run programs with graphics, so we'll set that up next.

Install the graphics software

The course staff has written a collection of Python functions that will let you easily draw graphics with your Python programs.

Download the qt library. Double-click on the downloaded file, which will start the installer and agree to all questions.

Download the pyside library, and double-click on the downloaded file. Agree to all questions, and click Next whenever you have the opportunity.

Now download cs1lib.py. You will be accessing this file a lot, so I suggest putting it in an easily accessed place, such as your Desktop. Drag cs1lib.py onto cs1proj in the PyDev Package Explorer tab of Eclipse:

Next, download eyes.py. Drag eyes.py from wherever you downloaded it to onto cs1proj in the PyDev Package Explorer tab of Eclipse:
