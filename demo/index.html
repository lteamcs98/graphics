A classroom in a browser

Interactive classroom is a web tool used to teach JavaScript programming through examples.

Grab the source from GitHub

 

Here's how it works

Easy Setup

Install our NodeJS application on your web server in a few simple steps.

Write a Challenge

Challenges consist of a function name, test input and a challenge solution. Construct them as JSON objects and upload them in batches to the website.

Distribute to Students

Embed the interactive editor associated with the challenge as an iframe on any web page.

View Results

Track students' progress and manage existing challenges through the application's GUI.

 
{
"title": "Find Maximum Value in a List",
"problem": "Write a function called findMax that returns the maximum in an array of integers.",
"functionNames": [
"findMax"
],
"inputArray": [
{
"functionName": "findMax",
"inputs": [
[ 1, 2, 5, 7, 0, 8, 10 ],
[ 169, 12, 4.3, 18, 3 ]
]
}
],
"outputArray": [
{
"functionName": "findMax",
"outputs": [ 10, 169 ]
}
]
}

Code Challenge Format

Code challenges are represented as JSON objects with the following fields.

  • title: The name of the challenge
  • problem: A description of the problem to solve
  • functionNames: Names of functions to define in challenge solution
  • inputArray: Array of inputs to test functions on
  • outputArray: Array of correct outputs corresponding to test input array
Our app iterates through an array of inputs for inputArray, so don't forget to encase the inputs in an array. For example, the example above shows two test arrays as the input. In order for the JSON to be read correctly, the two test arrays are nested in another array.
