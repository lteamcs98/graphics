SA 2: Draw a cartoon character

For this assignment, you will draw a cartoon character of your choosing using the graphics functions you've learned in class. You have full creative freedom in who you choose to draw and how you draw the character. Your effort is appreciated, but don't worry: you will not be graded on the basis of artistic ability. You can use the framework you developed in Short Assignment 1. Just make sure you copy the all the necessary files to a new directory, and replace animation.js from that assignment with the new JavaScript file that you will write. (All the calls to graphics functions should be added to this new JavaScript file.) Feel free to modify the size of the canvas as you see fit.
